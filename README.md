# Schedule Retweeter

## Requirements

- Node.js 12
- create Firebase project
- create [Secret Manager](https://cloud.google.com/solutions/secrets-management) keys
    - `twitter-consumer-key`
    - `twitter-consumer-secret`
    - `twitter-access-token-key`
    - `twitter-access-token-secret`
- setup `.firebaserc`

## Deploy

```bash
firebase deploy
```
